## 二、创新点定位

1. **骨架引导的自适应手绘电路接线算法**（核心，最独特）
   - Zhang-Suen骨架 → 路径验证 → 度约束 → 最近邻过滤
   - 解决了手绘图误差导致的过合并和漏连问题
   
2. **端到端手绘电路识别→接线→LLM评价**（系统完整性）
   - YOLO检测 + CRNN手写识别 + 接线算法 + 多LLM评价
   
3. **多LLM电路评审可信度分析**（应用创新）
   - DeepSeek vs GPT-4o vs Ollama vs 人工专家相关性验证

## 三、EI 期刊 — 3个月（概率85-90%）

### 第1-2周：标注 + 实验脚本

**标注**（最耗时，优先做）：

从 CGHD 选 **60-80 张**不同复杂度的图：
- 元件数 <5：15张
- 元件数 5-10：25张
- 元件数 10-20：25张
- 元件数 >20：15张
- 覆盖 10+ 位 CGHD 不同绘制者

标注工具：OpenCV + 鼠标回调，1天开发。每张图 5-15 分钟，总计 10-15 小时。

**实验脚本**（和标注并行）：
- `run_experiments.py`：自动跑 baseline / full / 5项消融，输出 CSV + 柱状图
- 支持批量调用 3 个 LLM API

**产出**：70 个 gt.txt + 实验对比表

### 第3-4周：跑实验 + LLM + 数据分析

| 实验 | 内容 | 耗时 |
|------|------|------|
| 接线准确率 | Full vs Baseline（纯P2J+JJ对齐） | 自动化，半天跑完 |
| 消融实验 | 去骨架/去度约束/去最近邻/去非对齐JJ/去Sobel | 自动化，半天跑完 |
| GNN对比 | 1个简单GNN接线baseline | 1周调包+跑 |
| LLM可信度 | 3 LLM × 2位专家 × 20张图 → Spearman | 专家评分需1-2周等待 |

**评估指标**：

| 指标 | 公式 | 含义 |
|------|------|------|
| 端口正确连接率 | TP端口 / 总端口 | 越高越好 |
| 误连率 | FP连接数 / 算法输出总连接数 | 越低越好 |
| 漏连率 | FN端口数 / 应连总端口数 | 越低越好 |
| 组准确率 | 完全正确的连通组 / 总连通组 | 越高越好 |

### 理论分析（+2周）

- 时间复杂度 O(WH + n²)
- 度约束+最近邻边数上界 ≤ 2n(k+2)，k=平均骨架分支数
- 对比无约束的 O(n²) 条边 → 解释了为什么算法能防止过合并
- 组合分析即可，不需要复杂数学证明

## 五、标注工具方案

```python
# 核心逻辑（~150行）

1. 加载图片 + YOLO检测结果
2. OpenCV显示：元件框+端口+标号
3. 鼠标回调：
   - 左键点击端口 → 选中（高亮）
   - 再点另一个端口 → 连线 → 存入当前组
   - 数字键 1-9 → 切换连通组（不同颜色）
   - S → 保存当前标注为 gt.txt
   - Backspace → 撤销上一条线
   - N → 下一张图，自动保存进度
```

开发 1 天，标注 70 张约 10-15 小时。
