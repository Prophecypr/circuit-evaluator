# 电路图智能评价系统 — 项目总结

## 一、项目概述

自动识别手绘/机画电路图 → 提取元件+数值+连线 → 生成 SPICE 网表 → Ngspice 仿真 → LLM 评价。

## 六、管线状态

| 环节 | 方案 | 状态 | 准确率 |
|------|------|:--:|:----:|
| 元件检测 | YOLOv8 | ✅ | 98.5% |
| 文字区域定位 | YOLOv8 | ✅ | 99.4% |
| 文字 OCR 读取 | text_annotations.json | ⚠️ | 仅限已有标注的图 |
| 接线点检测 | YOLOv8 | ⚠️ | 模型在单个元件上可用，全图不行 |
| 连线检测 | OpenCV Hough | ⚠️ | 能检测到线，但匹配不精确 |
| SPICE 网表生成 | 端口推断 | ⚠️ | 节点分配需改进 |
| Ngspice 仿真 | Ngspice | ✅ | 可运行 |
| LLM 评价 | DeepSeek V4 Pro | ✅ | 10/10 正确 |

# 4. YOLO 检测图像
